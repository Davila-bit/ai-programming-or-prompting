# User Manual
## Are We Still Programming, or Just Prompting the Machine?
### Interactive Presentation — Software Guide

---

## What This Software Is

An interactive web-based slide deck built with plain HTML, CSS, and JavaScript. All visuals are local SVG files. The deck is designed for a 16:9 screen and scales automatically to any browser window.

## Live Site

The presentation is hosted at:

https://davila-bit.github.io/ai-programming-or-prompting/

---

## Running Locally

1. Clone the repository: `git clone https://github.com/Davila-bit/ai-programming-or-prompting.git`
2. Open `index.html` in any modern browser (Chrome, Firefox, Edge, Safari).

No installation or internet connection required.

---

## Navigation Controls

| Action | Keys / Input |
|---|---|
| Next slide | `→` Right Arrow, `Space`, `Enter`, or click the right half of the screen |
| Previous slide | `←` Left Arrow, `Backspace`, or click the left half of the screen |
| Jump to first slide | `Home` |
| Jump to last slide | `End` |
| Toggle fullscreen | `F` |

The progress bar at the bottom shows the current slide number and title.
