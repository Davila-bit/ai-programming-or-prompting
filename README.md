# Are We Still Programming, or Just Prompting the Machine?

An interactive slide deck for a research presentation on AI-assisted coding, developer workflows, and what students should learn now.

**Live site:** https://davila-bit.github.io/ai-programming-or-prompting/

---

## Files

| File | Description |
|---|---|
| `index.html` | Slide deck — open this in a browser |
| `styles.css` | Visual theme |
| `app.js` | Navigation and scaling |
| `assets/` | SVG visuals |
| `ResearchPaper.pdf` | Accompanying IEEE paper |
| `UserManual.md` | Setup and hosting guide |

## Run Locally

Open `index.html` in any modern browser. No build step or server required.

## Navigation

| Input | Action |
|---|---|
| `→` / `Space` / `Enter` | Next slide |
| `←` / `Backspace` | Previous slide |
| `Home` / `End` | First / last slide |
| `F` | Fullscreen |
